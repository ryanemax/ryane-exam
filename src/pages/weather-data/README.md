# 项目信息
- 天气信息发布
    - Weather Data
- 功能描述
    - 天气发布信息管理模块，主要功能是对一周天气数据进行增删查改操作
    - 基于WeatherData特性模块，对天气信息数据进行筛选分析
- NG及TS依赖
    - MatButtonModule官方提供MD组件库 
    - PipesModule 本项目的自定义管道
    - DirectivesModule 本项目的自定义指令
    - Parse 后端数据访问库，cloud/cloud.ts
- ES4/5依赖
    - echarts.js 【ES4/5】旧版JS第三方库，用于图表绘制

- 目录结构(pages)
    - weather-data/
        - weather-data-list
        - weather-item
