# 项目信息
- 相关数据库集合
    - Coffee-star1
- 功能描述
    - 咖啡管理模块，主要功能是对咖啡数据进行增删查改操作
    - 基于Coffeestar1特性模块，对咖啡数据进行筛选分析
- NG及TS依赖
    - MatButtonModule官方提供MD组件库 
    - PipesModule 本项目的自定义管道
    - DirectivesModule 本项目的自定义指令
    - Parse 后端数据访问库，cloud/cloud.ts
- ES4/5依赖
    - echarts.js 【ES4/5】旧版JS第三方库，用于图表绘制

- 目录结构(pages)
    - coffee-star1/
        - coffee-star1-detail
        
        - coffee-star1-edit
